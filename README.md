Random Quote Generator

A simple and elegant web app that displays random inspirational quotes using an API.  
Built with HTML, CSS, and JavaScript, this project demonstrates API fetching, DOM manipulation, and basic web styling.

---

Features
- Fetches random quotes from an external API (`https://api.quotable.io/random`)
- Displays the quote and its author dynamically
- “New Quote” button to load a new random quote instantly
- “Tweet” button to share the current quote directly on Twitter
- Simple and clean UI 

---

Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
